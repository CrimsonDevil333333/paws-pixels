## ğŸš€ Getting Started

### Prerequisites

- Node.js 24+
- pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/CrimsonDevil333333/paws-pixels.git
   cd paws-pixels
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Run the development server:
   ```bash
   pnpm next dev
   ```

4. Build for production:
   ```bash
   pnpm build
   pnpm start
   ```
